Lyrics Plugin for Windows Media Player 是一个为Windows Media Player 添加显示歌词的插件,不过对于中文歌曲应该没什么效果!

<!--more-->

Windows Media Player 其实是有内建字幕和歌词的功能的，只不过仅支持它自己独有的格式，所以要想显示歌词，只能通过安装插件的办法。
